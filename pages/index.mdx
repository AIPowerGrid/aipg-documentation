# AI Power Grid

**Free AI for everyone.** Build agents, ship products, learn AI — without barriers.
The Grid is a decentralized GPU network with an OpenAI-compatible API, funded by
paid users so anyone can build for free.

<Callout type="info">
  Free daily access is live — no credit card, no trial that expires.
  [Get your free API key](https://api.aipowergrid.io/register) to start building.
</Callout>

---

## How It Works

Every paid request funds free access for someone else. The more the network grows,
the more free AI it can provide.

```
   Paid Users  ─funds→  Node Operators  ─compute→  Free Users
       │                       │                        │
   APIs &                   Earn AIPG               Learn,
   integrations             per request             build, ship
```

| Role | What you do | What you get |
|------|-------------|--------------|
| **Free user** | Sign up, get an API key, start building | Daily free quota, OpenAI-compatible API |
| **Paid user** | Top up for higher limits | Unlimited access, your usage funds free tier |
| **Node operator** | Run a GPU worker | Earn AIPG per request served |
| **Validator** _(soon)_ | Verify inference results | Earn verification fees, secure the network |

## Why AIPG?

- **Free tier that doesn't expire.** Students, hobbyists, devs in emerging markets — everyone gets daily access.
- **OpenAI-compatible.** Drop-in replacement for `/v1/chat/completions`. Existing SDKs just work.
- **No gatekeepers.** No corporate API. No usage profiling. No model deprecations on someone else's schedule.
- **Community-owned compute.** Anyone with a GPU can join the network and earn.
- **Heading on-chain.** Validators, smart-contract payments, and confidential computing in active development.
