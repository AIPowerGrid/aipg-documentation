# AI Power Grid

Decentralized AI inference on Base. Generate images, chat with LLMs, and build on The Grid.

---

## What is AIPG?

AI Power Grid is a decentralized network where GPU owners run AI models and users generate content. No corporate API. No centralized control. Just open-source AI powered by the community.

**The Grid** = the network of workers, models, and infrastructure that makes it all work.

## How It Works

```
┌─────────────────────────────────────────────────────────┐
│                       THE GRID                          │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   USERS              WORKERS              OUTPUT        │
│                                                         │
│   Submit       →     GPUs run        →    Images,       │
│   prompts            AI models            text, etc.    │
│                                                         │
│   Pay with Den       Earn AIPG            Verified      │
│   (電) credits        rewards              on-chain      │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

1. **You request** - Submit a prompt via our apps or API
2. **Workers generate** - Decentralized GPU nodes run the models
3. **You receive** - Get your AI-generated content
4. **Workers earn** - GPU operators earn AIPG for their compute
