# For Developers

Build on The Grid. Access decentralized AI inference, launch tokens, and integrate with the AIPG ecosystem.

---

## Quick Start

1. **Get an API key** at [api.aipowergrid.io/register](https://api.aipowergrid.io/register)
2. **Call the streaming API** — it's OpenAI compatible

```python
from openai import OpenAI

client = OpenAI(
    base_url="https://api.aipowergrid.io/v1",
    api_key="your-grid-api-key",
)

stream = client.chat.completions.create(
    model="grid/openai/gpt-oss-120bb",
    messages=[{"role": "user", "content": "What is AI Power Grid?"}],
    stream=True,
)
for chunk in stream:
    print(chunk.choices[0].delta.content or "", end="")
```

That's it. Any tool that works with OpenAI works with The Grid.

## API Endpoints

### Streaming API (New)

Real token-by-token streaming. **[Full docs →](/streaming-api)**

| Endpoint | Format | Description |
|----------|--------|-------------|
| `POST /v1/chat/completions` | OpenAI | Text gen with streaming |
| `POST /v1/messages` | Anthropic | Text gen with streaming |
| `POST /v1/images/generations` | OpenAI | Image generation |
| `GET /v1/models` | OpenAI | List available models |
| `GET /v1/health` | Grid | Worker status |

**Authentication:** `Authorization: Bearer YOUR_KEY` or `apikey: YOUR_KEY`

### Legacy API (v2)

Poll-based generation. Still supported for backward compatibility.

| Endpoint | Description |
|----------|-------------|
| `POST /api/v2/generate/text/async` | Submit text generation |
| `GET /api/v2/generate/text/status/{id}` | Poll for text results |
| `POST /api/v2/generate/async` | Submit image generation |
| `GET /api/v2/generate/status/{id}` | Poll for image results |
| `GET /api/v2/status/models` | List all models |

**Authentication:** `apikey: YOUR_KEY` header

**Swagger docs:** [api.aipowergrid.io/api](https://api.aipowergrid.io/api)

## SDK

### Grid SDK (Python)

The official Python SDK with streaming support.

**Repository:** [github.com/AIPowerGrid/grid-sdk](https://github.com/AIPowerGrid/grid-sdk)

```bash
pip install grid-sdk
```

```python
from grid_sdk.streaming import StreamingChatClient

client = StreamingChatClient(
    base_url="https://api.aipowergrid.io/v1",
    api_key="your-key",
)

# Streaming
for chunk in client.stream_chat(
    model="grid/openai/gpt-oss-120bb",
    messages=[{"role": "user", "content": "Hello!"}],
):
    print(chunk, end="")

# Async streaming
async for chunk in client.astream_chat(
    model="grid/openai/gpt-oss-120bb",
    messages=[{"role": "user", "content": "Hello!"}],
):
    print(chunk, end="")
```

### OpenAI SDK (Any Language)

Since the API is OpenAI compatible, any OpenAI SDK works:

**Python:**
```bash
pip install openai
```

```python
from openai import OpenAI
client = OpenAI(base_url="https://api.aipowergrid.io/v1", api_key="your-key")
```

**JavaScript/TypeScript:**
```bash
npm install openai
```

```typescript
import OpenAI from 'openai';
const client = new OpenAI({ baseURL: 'https://api.aipowergrid.io/v1', apiKey: 'your-key' });
```

**curl:**
```bash
curl -N https://api.aipowergrid.io/v1/chat/completions \
  -H "Authorization: Bearer YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{"model":"grid/openai/gpt-oss-120bb","messages":[{"role":"user","content":"Hi"}],"stream":true}'
```
