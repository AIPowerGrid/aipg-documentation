## Token Utility

AIPG is the native token of The Grid. It has three core uses:

### 1. Worker Rewards

GPU operators earn AIPG for running AI inference on The Grid. Rewards scale with:
- Model size (bigger models = more AIPG per job)
- Output tokens generated
- Context length processed
- Uptime and reliability

### 2. Staking

Stake AIPG in the [StakingVault](/staking) to earn passive rewards. No lockup — withdraw anytime. Rewards are funded by the treasury and distributed proportionally.

**StakingVault:** [`0x3ED14A6D5A48614D77f313389611410d38fd8277`](https://basescan.org/address/0x3ED14A6D5A48614D77f313389611410d38fd8277)

### 3. Worker Bonding

Workers can optionally bond AIPG for:
- Higher priority job routing
- Increased trust score on the network
- Access to premium job tiers (future)

Bonding is not required to earn — it's a signal of commitment.

## Den (電) — Platform Credits

Den is a separate priority system used for queuing AI generation requests.

| Property | Value |
|----------|-------|
| **Purpose** | Request priority in the generation queue |
| **Earned by** | Running workers, signing up |
| **Spent on** | Image, text, and media generation |
| **Monetary value** | None — den is not a currency |
| **Transferable** | Yes, between accounts |

Users with more den get served faster. Workers accrue den for completed jobs proportional to the compute provided.

Den is **not** AIPG. It's an internal credit system for fair resource allocation.

## History

AIPG originally launched as a Ravencoin fork with Proof of Work mining and a halvening emission schedule. The project has since migrated to **Base L2** with all 150M tokens minted as a fixed-supply ERC-20. The original PoW chain is no longer active.

For the original whitepaper describing the PoW era, see [Whitepaper](/whitepaper).
